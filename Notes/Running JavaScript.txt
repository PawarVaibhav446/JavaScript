
Where can we run JavaScript?

=> 1) In a HTML file under <script> </script> tag.
    The <script> tag is used to embed a client-side script (JavaScript).
    The <script> element either contains scripting statements,
    or it points to an external script file through the src attribute. 

=> 2) In a HTML file we can link Js File with the help of script tag.
    Example: <script src = ./index.js> </script> 

=> 3) We can Code JavaScript in a Browser's Console.

=> 4) With the Help of Node.js, we can run JavaScript in Node terminal
     Example: node FileName.js
